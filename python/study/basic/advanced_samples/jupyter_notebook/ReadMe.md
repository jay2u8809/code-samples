## Install Jupyter Notebook
- Jupyter Notebook을 설치하면 IPyhton까지 함께 설치된다.
```shell
    $> pip install jupyter
```

## How to Use IPython
```shell
    $> ipython
```

## How to Use Jupyter Notebook
```shell
    $> jupyter notebook
```
### 파이썬2와 파이썬3를 동시에 실행하기
```shell
    $> nohup jupyter notebook &
```
- 위의 명령어를 한번 실행한 후 **jupyter notebook** 명령어를 실행하면 동시에 사용할 수 있다.
